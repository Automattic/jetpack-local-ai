## Setup of dev environment

1. `nvm use`

3. `npm run build`
4. `npm run wp-env start`
5. http://localhost:8903/wp-admin/post-new.php
6. Write some content
7. Navigate to "AI Excerpt" in sidebar
8. Click "Generate"
9. See the excerpt being generated
